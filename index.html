What is a regular language?

A regular language is any language that can be recognized by a finite automaton — a machine with a fixed, finite amount of memory. Think of it as a pattern a very simple machine can check without needing to count.

Examples of regular languages: all binary strings ending in 00  |  all strings starting with 'ab'  |  all strings over {a,b} with even length.

But not every language is regular. Some patterns require remembering an arbitrary amount — like matching parentheses or counting equal numbers of letters. How do we prove a language is NOT regular?

That's exactly what the Pumping Lemma is for. It's a proof tool — a contradiction machine — used to show that certain languages exceed what finite automata can handle.

The pumping length p

Every regular language has a special constant called the pumping length p. It's related to the number of states in the finite automaton that recognizes the language.

Think of p as a "memory limit". If a string is at least as long as p, the automaton must have visited some state twice while reading it — which means there's a loop in the computation.

This repeated state is the core insight. When the machine loops through the same state, we can repeat (pump) the part of the string that caused the loop, and the machine will still end in the same accepting or rejecting state.

If |w| ≥ p, a loop exists in the automaton's path through w

Splitting the string: x, y, z

Any string w with |w| ≥ p can be divided into three consecutive parts: w = xyz. Each part plays a specific role in the lemma.

x — prefix
The unchanging part before the loop. x can be empty (|x| ≥ 0).
y — the pump
The repeatable middle part. Must be non-empty: |y| ≥ 1. This is the loop body!
z — suffix
The unchanging tail after the loop. z can be empty (|z| ≥ 0).
Key constraint: |xy| ≤ p — the pump y must occur within the first p characters. This ensures y falls inside the looping region of the automaton.

Pumping — repeating y

Here's the power move: for any regular language, you can repeat y any number of times (including zero) and the resulting string must still be in the language.

xyiz ∈ L   for all i ≥ 0

This means all of these pumped strings must belong to L:

i=0: xεz     (y deleted entirely)
i=1: xyz     (original string w)
i=2: xyyz    (y doubled)
i=3: xyyyz   (y tripled)
If any single pumped string falls outside the language L — then L cannot possibly be regular. We have our contradiction!

The pumping lemma (formal)

Putting it all together, here is the official statement:

If L is a regular language, then
∃ p ≥ 1  (pumping length)  such that
∀ w ∈ L  with |w| ≥ p,
∃ x, y, z  with w = xyz  such that
  (1)  |y| ≥ 1
  (2)  |xy| ≤ p
  (3)  ∀ i ≥ 0, xyiz ∈ L
Important: The lemma is a one-way implication. Regular → pumping holds. But pumping holding does NOT mean the language is regular. It's only used to prove NON-regularity (by contradiction).

Proof by contradiction

To prove L is NOT regular, we show the pumping lemma conditions cannot all hold for L. The structure of the proof is always the same:

01
Assume L is regular. Then a pumping length p exists.
02
You choose a specific string w ∈ L with |w| ≥ p. Choose strategically!
03
The adversary splits w = xyz with |y| ≥ 1 and |xy| ≤ p. They pick — you must handle ALL valid splits.
04
You show that for every valid split, some xyiz ∉ L. This violates condition (3).
05
Contradiction: L cannot be regular. ∎
It's a game: you pick the string, the adversary picks the split, and you must always find a pumped string outside L. If you can always win, L is not regular.

Classic example: {aⁿbⁿ | n ≥ 0}

The language of equal a's followed by equal b's is the most famous non-regular language. Let's prove it step by step.

Setup: Assume {aⁿbⁿ} is regular → ∃ pumping length p. Choose w = apbp. Then |w| = 2p ≥ p, so the lemma applies.
Adversary's split: Any xyz with |xy| ≤ p and |y| ≥ 1. Since xy is within the first p characters (which are all a's), y must consist entirely of a's. Write y = ak for some k ≥ 1.
xy²z = ap · ak · bp = ap+kbp   (more a's than b's!)
Contradiction: xy²z has p+k a's but only p b's, so xy²z ∉ {aⁿbⁿ}. This violates condition (3). Therefore {aⁿbⁿ} is NOT regular. ∎
